# Assembly Language Assignments - Week 1

This repository contains the source code (`.asm`) and compiled files (`.obj`, `.exe`) for the Week 1 Assembly Language assignments.

> **⚠️ Compatibility Note:** > Currently, these exercises and pre-built executables are designed to run exclusively on **Windows 64-bit** environments. Support for additional operating systems and architectures is planned for future updates.

## 📁 Repository Structure

* **TestASM**: Environment and basic syntax check program.
* **Week1_Ex1**: Exercise 1.
* **Week1_Ex2**: Exercise 2.
* **Week1_Ex3**: Exercise 3.
* **Week1_Bonus**: Advanced/Bonus exercise.

*Each exercise consists of 3 file formats:*
* `.asm`: Assembly source code file.
* `.obj`: Object file (generated after assembling the source code).
* `.exe`: Executable file (generated after linking the object file).
